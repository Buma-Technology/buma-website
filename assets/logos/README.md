# Client logos

Drop client logo files here. Once the files are in, the text brand names in the
logo bars / rosters (home, about, work) get swapped for these images, rendered
**monochrome** (uniform dark, lifting to full strength on hover) so the bars stay
visually consistent.

## Specs

- **Format:** SVG preferred. Otherwise transparent **PNG**, at least ~400px wide.
- **Background:** must be **transparent** — supply the logo mark/wordmark only, not
  a colored tile or badge background. (A logo baked onto a red/white rectangle can't
  be cleanly recolored to monochrome.)
- **Filename:** lowercase, hyphenated slug, exactly as listed below.
